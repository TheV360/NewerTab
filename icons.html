Icons

They're a mess.

Later, I'll add an easy icon UI, but for now you're stuck with a single text box. What URL do you put in the box? Let's find out.


Font Awesome icons

The best place to get icons is from Font Awesome icons. They have a bunch of free icons and logos you can use easily. To use their icons, first open the cheatsheet and find an icon. Then, remember its name and find what section it's under.

  • If it's under brand icons, then write img/brands.svg in the text box.
  • If it's under regular icons, then write img/regular.svg in the text box.
  • If it's under solid icons, then write img/solid.svg in the text box.

After that file name in the textbox, add a hashtag and the name of the icon you chose, like so: (don't add a space in-between!)

img/brands.svg#reddit-alien

Your icon should work now.


Everything else

You can also just put any .svg URL in and it'll work. Even data URIs will work.
