Later, I'll add an easy icon UI, but for now you're stuck with a single text box. What URL do you put in the box? Let's find out.

Font Awesome icons

To use Font Awesome icons, first open the cheatsheet and find an icon you like. Write down its name and find what section it's under.

  • If it's under brands, then write img/fa-brands.svg in the text box.
  • If it's under regular, then write img/fa-regular.svg in the text box.
  • If it's under solid, then write img/fa-solid.svg in the text box.

After that file name in the textbox, add a hashtag and the name of the icon you chose, like so: img/fa-brands.svg#reddit-alien

Your icon should work now.
